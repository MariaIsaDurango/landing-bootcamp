# ☕ MIDURANGO Café Bootcamp

¡Bienvenida a la combinación perfecta entre cafeína y código! Esta es la **Landing Page oficial** de un bootcamp intensivo diseñado para transformar ideas en proyectos digitales reales.

> "Aprende programación mientras disfrutas café."

## 🎨 Características del Proyecto
- **Diseño Temático:** Paleta de colores inspirada en el café (#8B4513, #E6CDB4).
- **Tipografía Moderna:** Uso de la fuente profesional `Bungee` vía Google Fonts para títulos impactantes.
- **Interfaz Responsiva:** Estructura limpia con secciones de beneficios y llamados a la acción (CTA).
- **Efectos Visuales:** Sombras suaves (`box-shadow`) y filtros en el logo para mayor profundidad.

## 🛠️ Tecnologías Utilizadas
*   **HTML5:** Estructura semántica completa.
*   **CSS3:** Estilos personalizados, efectos `:hover` y posicionamiento absoluto.
*   **Google Fonts:** Integración de fuentes externas.
*   **Git & GitHub:** Control de versiones y despliegue.
